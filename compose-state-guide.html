📦 State — armazenamento
mutableStateOf base
Use quando precisa de qualquer estado reativo. Fundação de tudo.
ver exemplo
// sempre dentro de remember var nome by remember { mutableStateOf("") } // ou sem delegate val nome = remember { mutableStateOf("") } nome.value = "Fabiano"
rememberSaveable persistência
Use quando o estado deve sobreviver à rotação ou process death.
ver exemplo
// primitivos — funciona direto var texto by rememberSaveable { mutableStateOf("") } // classe própria — use @Parcelize @Parcelize data class User(val name: String) : Parcelable var user by rememberSaveable { mutableStateOf(User("")) } // tipo de lib — use listSaver val Saver = listSaver( save = { listOf(it.name, it.age) }, restore = { User(it[0], it[1]) } )
mutableStateListOf listas
Use para listas mutáveis onde add/remove devem atualizar a UI.
ver exemplo
// ❌ List normal — Compose não detecta mudança var items by remember { mutableStateOf(mutableListOf()) } items.add("X") // UI não atualiza! // ✅ SnapshotStateList — detecta tudo val items = remember { mutableStateListOf("A", "B") } items.add("C") // UI atualiza ✓ // sempre use key no LazyColumn! LazyColumn { items(items, key = { it }) { Text(it) } }
derivedStateOf derivado
Use quando um valor muda menos que o estado original. Evita recompositions.
ver exemplo
// ❌ Recompõe a cada pixel de scroll val show = scrollState.value > 100 // ✅ Recompõe só quando bool muda val show by remember { derivedStateOf { scrollState.value > 100 } }
produceState async → state
Converte suspend / callback / Flow em State. LaunchedEffect que retorna valor.
ver exemplo
val user by produceState<User?>( initialValue = null, key1 = userId ) { value = repo.getUser(userId) // suspend // ou com cleanup (callback legado) val listener = Listener { value = it } manager.register(listener) awaitDispose { manager.unregister(listener) } }
snapshotFlow state → flow
Converte State em Flow para usar operadores como debounce e filter.
ver exemplo
LaunchedEffect(Unit) { snapshotFlow { searchQuery.value } .debounce(300) .filter { it.length > 2 } .collect { query -> viewModel.search(query) } } // caso Maps — evita jank snapshotFlow { cameraState.position } .debounce(500) .collect { viewModel.loadMarkers(it) }
⚡ Effects — efeitos colaterais
LaunchedEffect coroutine
Coroutine atrelada ao ciclo de vida. Reinicia quando a key muda.
ver exemplo
// executa quando userId muda LaunchedEffect(userId) { user = repo.getUser(userId) } // executa uma vez (Unit = sem mudança) LaunchedEffect(Unit) { viewModel.init() } // cancela automaticamente ao sair da tela
rememberCoroutineScope evento
Coroutine disparada por evento do usuário (clique, swipe).
ver exemplo
val scope = rememberCoroutineScope() Button(onClick = { scope.launch { // assíncrono api.salvar(dados) } }) { Text("Salvar") } // scope é cancelado quando // o Composable sai da tela
SideEffect síncrono
Executa após cada recomposition. Sincroniza com código não-Compose.
ver exemplo
// síncrono — não use suspend aqui SideEffect { analytics.setScreen(screenName) firebaseLogger.setUser(userId) } // roda após TODA recomposition bem-sucedida // use só para código rápido e não-suspend
DisposableEffect cleanup
Registrar/desregistrar listeners e callbacks. Exige onDispose.
ver exemplo
DisposableEffect(Unit) { val listener = LocationListener { onLocation(it) } manager.register(listener) onDispose { // obrigatório! manager.unregister(listener) } }
@Immutable / @Stable performance
Evita recompositions desnecessárias em classes customizadas.
ver exemplo
// ❌ Compose não confia — recompõe sempre data class User(val name: String) // ✅ @Immutable — todos val, Compose confia @Immutable data class User(val name: String) // ✅ @Stable — var com mutableStateOf @Stable class UserState(name: String) { var name by mutableStateOf(name) } // ⚠️ Lambda em data class quebra @Immutable // Use slot pattern em vez disso
graphicsLayer vs drawBehind fases
graphicsLayer transforma. drawBehind desenha. Ambos diferem leitura para Drawing.
ver exemplo
// graphicsLayer → transforma via GPU Modifier.graphicsLayer { alpha = scrollState.value / 100f scaleX = 1.2f rotationZ = 45f } // drawBehind → canvas custom atrás do conteúdo Modifier.drawBehind { drawCircle(Color.Blue, radius = size.minDimension) } // ambos: leitura ocorre em Drawing, não Composition // → sem recomposition desnecessária
